Wind speed profile at turbine hub height

This tool calculates the wind speed profile at turbine hub height.

I/O Parameters

This tool requires a Sentinel 1 Level 2 OCN source product.

Processing Parameters

Wind Height
This parameter specifies the turbine hub height. The wind speed profile is calculated for this height.

Shear exponent
Examples of shear exponent values are:

  • Unstable air above open water surface: 0.06
  • Neutral air above open water surface: 0.10
  • Unstable air above flat open coast: 0.11
  • Neutral air above flat open coast: 0.16
